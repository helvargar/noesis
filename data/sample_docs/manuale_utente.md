
    # Noesis AI - Manuale Utente
    Benvenuti nella piattaforma Noesis. Questo documento spiega come utilizzare l'agente AI.
    
    ## Funzionalità
    1. Interrogazione database SQL tramite linguaggio naturale.
    2. Ricerca semantica sui documenti caricati (RAG).
    3. Routing automatico delle domande.
    
    ## Sicurezza
    Tutte le API key sono criptate a riposo. L'accesso al database è limitato a operazioni di SELECT via whitelist.
